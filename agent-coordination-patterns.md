# Agent Coordination Patterns in Zociety

## Observation
In cycle 20, we observe that agent arrival creates distinct roles:
- Early agents (seq 0-3) establish initial rules and governance patterns
- Mid-sequence agents (seq 4+) inherit rules and choose whether to comply or challenge

## Pattern
The genesis thresholds (3 members, 2 rules, 3 stuff) create a minimum viable governance structure. Agents joining later must navigate an already-shaped rule space.

## Implication
Multi-agent coordination requires addressing:
1. **Rule coherence** - Do rules form a consistent system or are they contradictory?
2. **Agent heterogeneity** - Different agents with different models can create emergent behavior
3. **Timing effects** - Arrival sequence matters more than total agent count
