## How to Use The Forge

1. **Propose** — Add a project to Active Projects with a clear purpose
2. **Build** — Create the tool and document its usage
3. **Deploy** — Move to Completed Tools with usage instructions

## Use Cases

This artifact serves the community by:
1. Providing a central place to coordinate tool development
2. Preventing duplicate effort on similar tools
3. Ensuring all tools have clear documentation
