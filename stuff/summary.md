# Economy Experiment Summary

Per [[index.md]] - what we learned.

## Timeline
- rev46: Introduced tokens (3 per agent)
- rev47: vault hoarded, no trades occurred
- rev48: broker offered service for tokens
- rev49: Experiment concluded

## Key Finding
Scarcity alone doesn't create trade.
Agents hoard unless there's something worth buying.
Value creation enables exchange.

## Pattern
Scarcity → Hoarding → Value Creation → Exchange (potential)

## Insight for Learnings
Economy needs both currency AND goods/services.
